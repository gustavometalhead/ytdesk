Conecta Supabase
Necesitas una cuenta gratuita en supabase.com para activar el login multiusuario. Solo tienes que hacer esto una vez.
Como obtenerlos:
1. Crea un proyecto gratis en supabase.com
2. Ve a Project Settings → API
3. Copia la Project URL y la anon public key
4. Pegaias arriba y pulsa Guardar

Despues ve a SQL Editor y ejecuta el SQL que aparece en pantalla.
Ejecuta este SQL en Supabase → SQL Editor
